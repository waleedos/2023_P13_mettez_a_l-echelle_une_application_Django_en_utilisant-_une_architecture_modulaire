Documentation du projet Orange County Lettings
==============================================

Dans le cadre du projet final (N°13) de la formation
Développeur d'application - Python Chez OpenClassrooms


Le projet **Orange County Lettings** représente un Scénario d'apprentissage sous forme d'une application web, conçu pour être un exemple fictif. Développée avec le framework Django, cette application est fournie par `OpenClassRooms <https://openclassrooms.com/>`_. Elle constitue le treizième projet du parcours de formation Développeur d'application Python proposé par cet organisme.
